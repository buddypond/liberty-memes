# $LIBERTY Memes

# Requirements

*Any* HTTP Server that can server STATIC HTML

**EASY MODE**

## Try `vite` and `npm` if needed

```
git clone https://github.com/buddypond/liberty-memes.git
npm install vite -g
cd liberty-memes
open public/memes
vite dev
```

Now open `http://localhost:5173`
